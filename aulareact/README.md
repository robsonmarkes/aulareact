## server-api - Config da base de dados mysql 

Basta trocar o arquivo conn.js que se encontra no caminho server-api/conn.js

```json
	{
		host: 'localhost', //host
		port: '3306', //porta mysql
		user: 'root', //user
		password: '', //password
		database: 'nomedb' //nome da base 
	}
```
